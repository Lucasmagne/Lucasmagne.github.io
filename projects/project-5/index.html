Dreamscapes

A short action game designed to help stressed teens reset, breathe, and return to work more focused.

Introduction

Sometimes, teens feel stressed after so much work and want a break, and a short game session would definitely help get you back into a productive state and refocus your stress. From a study from ASU, 61% of teens who are stressed in 2025 claim to feel more stressed after playing games because games feel infinite with no end, and the adrenaline does not stop. We can help with this problem with our game Dreamscapes. This is a quick, action packed game to help teens escape their stress momentarily so they can come back more refreshed and ready to work. Dreamscapes is a simulation game where you are trying to escape the monsters. It will go from a very anxious and busy environment to a more calm and relaxing environment at the end. The point of this game is to refocus their stress on the game without having users become addicted so they can still work on what they need to do.

Methodology

Using Unity as the game engine, Visual Studio Code for the code in C#, and Procreate and Photoshop for the character art, animations, and background, we create Dreamscapes. This game includes quick gameplay, and exhilarating action that slows and calms down to help the player relax and boost productivity after their tightened state.

Design

This game is playable in approximately 10 minutes, a perfect amount of time for users to relax and refocus their stress onto the game so that they can get back to their work in a better mental state. In the game, the user starts in a rocky, dark scene, mirroring their mental state at the time due to the stress. They are chased around a world with randomly generated obstacles that they must jump over, with a monster chasing them. As they progress, the scenery will become more serene, leading them into a more calm state to get back to work. Unlike traditional games designed for extended engagement, Dreamscapes intentionally limits replayability through a cooldown system, encouraging users to return to their responsibilities after relaxing. This game doesn't get boring either, as the obstacles are randomly generated so there will always be a new path for users to traverse.
